AjfRepeatingNode

Interface that is extended to represent a Repeating Node. Repeating nodes are used any time a node can or must be repeated N times, where N is the value of the "formulaReps" property.

Properties
Name Description

disableRemoval: boolean

If true show all slides of repeating node in readonly mode. Except the last slide. In the last slide remove button is always disabled and add button is enabled only when the last slide is valid.

formulaReps: AjfFormula

Is the number of node repetition Example: "opd_treatment == 'Yes' && ($groupReps || 1) || 0"

maxReps: number

Max number of repetitions

minReps: number

Min number of repetitions

AjfBaseNodeInstance

The instance of an ajfNode. A single ajfNode can have multiple instances with different values differentiated by a suffix, when used in a repeating slide. (eg. country__0 = 'ITA', country__1 = 'ESP' city__0__0 = 'Milano', city__0__1 = 'Roma', city__1__0 = 'Madrid' )

The AjfNodeInstance visibility is determined by the visibility of the current branch and the visibility condition of the node itself.
