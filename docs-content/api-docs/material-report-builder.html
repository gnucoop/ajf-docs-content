instantColumnValue

This method assigns the new width to the idx column and recalculates the width of the remaining columns of the layout. The range value are from 0,1 to 1.

RULES: The min value for column is 0,1. The sum of all columns width is always 1. The method round the values. If is present only one column the width is always 1.

When the new value > old value: the width of the remaining columns decreases. When the new value < old value: the width of the remaining columns increases.

When values ​​are periodic, rounding assigns the gap to the current value. For example: 3 columns with 0,33 believe 1 column 0,34 and 2 columns 0,33.
